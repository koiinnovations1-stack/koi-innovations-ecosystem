III. Typography

Type System

Font Families

Serif
Cormorant Garamond
The elegant foundation
Use for: Headers, titles, manifesto-weight text, serif accents, emphasis
Weights: 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)
Style: Roman (upright) and Italic. Italic = elegant, doctrine-weight
Character: Sophisticated. Literary. Premium. Carries authority without coldness.
Sans-Serif
Inter
The modern clarity
Use for: Body text, navigation, UI elements, labels, operational text
Weights: 300 (Light), 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)
Character: Modern. Clean. Corporate. Highest legibility at all sizes.

Type Scale

Display (H1)
48px–120px · Cormorant · Bold · For page titles
Headline (H2)
32px–64px · Cormorant · SemiBold · For section titles
Subheading (H3)
24px–32px · Cormorant · Medium · For subsections
Body
16px–17px · Inter · Regular · For running text · 1.8x line-height
Label
11px–12px · Inter · SemiBold · For labels, metadata · 3–4px tracking
IV. Brand Voice

Three Voice Tiers

Architect Voice
Use for systems, governance, operations, infrastructure, B2B communications.
Tone: Precise. Disciplined. Clear. Direct without coldness.
Language: Active voice. Declarative. No hedging.
"Standards win. Systems scale."
Atelier Voice
Use for luxury, craft, bespoke, premium positioning, storytelling.
Tone: Refined. Intentional. Sophisticated. Inviting without icy.
Language: Sensory detail. Metaphor. Narrative. Craft as spiritual practice.
"The work refines the people. The people refine the work."
Host Voice
Use for community, gathering, discovery, culture, experience.
Tone: Warm. Inclusive. Generous. Presence-based.
Language: Second person ("you"). Questions. Invitation. Belonging language.
"Better is possible. Better is built together."
V. Usage Rules

Standards & Guardrails

Color Usage
Koi Gold: Never more than 30% of a layout. Accent, not dominant.
Koi Black: Primary background in dark sections. Primary text in light sections.
Ivory: Light background in light sections. Text color on dark.
Venture colors: Confined to their own venture contexts. No cross-bleeding.
Typography Usage
Pattern: Serif headers + sans-serif body. This is locked. Do not reverse.
All caps: Only for labels, metadata, navigation. Never for body copy.
Italics: Use sparingly. Cormorant Italic for emphasis, quotes, taglines.
Bold: For impact statements, key phrases, decision anchors.
Voice Mixing
One voice per document. Pick the right voice for the audience and stick with it. Architect = operational/internal. Atelier = luxury/craft. Host = community/culture.
Cross-Venture Rule
Each venture is sovereign in its own materials. Do not blend venture aesthetics. Exception: Master framework documents reference all ventures but maintain Koi Black + Gold + Ivory palette.
