IV. Pricing Rules & Guidelines

How We Quote & Bill

Rule 1: Lead with Scope, Not Per-Window Rate
Client-facing language: "Window service starts at $175, customized based on access, pane detail, and service level." Never lead with "$12 per window" — this invites price shopping. Lead with minimum and scope.
Rule 2: Add-Ons Are Non-Negotiable
Tracks, sills, screens, hard water = separate line items. These are NOT included in base service. Quote explicitly. No surprises.
Rule 3: Double Units Count as Two Windows
Double-hung or double-width windows = 2 windows. Not 1. Each pane system counts.
Rule 4: Hard Water Stains Are Assessed On-Site
Pricing depends on severity. Light ($10), moderate ($15), heavy ($25). Assess before quoting. Document in scope.
Rule 5: Volume Discounts Apply at 15+ Windows
15+ windows = $1/window discount. 30+ windows = $2/window discount. Applied automatically, mentioned in quote.
Rule 6: Minimums Are Hard Floors
Residential minimum $175. Commercial minimum $200. No exceptions. Travel + setup always included.
